## Features

- **Responsive Design**  
  Fully optimized for desktops, tablets, and mobile devices.

- **Smooth Navigation**  
  Custom JavaScript implementation for smooth scrolling between sections.

- **Dynamic UI**  
  Moving gradient background and interactive hover effects on service cards.

- **Service Showcase**  
  Covers:
  - Flutter Development  
  - Full Stack Development  
  - Artificial Intelligence  
  - Data Science  

- **Active Link Highlighting**  
  Navigation bar updates automatically based on scroll position.

---

## Tech Stack

- **HTML5** – Semantic and structured markup  
- **CSS3** – Flexbox, Grid, Animations  
- **JavaScript (Vanilla)** – Scroll logic & interactivity  
- **Font Awesome** – Icons and visuals
